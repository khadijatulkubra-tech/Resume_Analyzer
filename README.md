# Resume_Analyzer
An NLP-based Flask web application that analyzes resumes, extracts skills, scores CVs, and provides improvement suggestions.

---

## 📌 Project Description
This is a Flask-based Resume Analyzer that extracts text from resumes (PDF, DOCX, TXT) and evaluates them based on skills, keywords, contact details, and resume structure.

---

## 🚀 Features
- Upload Resume (PDF, DOCX, TXT)
- Extract text automatically
- Skill detection (Programming, Web, AI, etc.)
- Education, Experience & Certification detection
- Contact info check (Email, Phone, LinkedIn, GitHub)
- Resume scoring system (0-100)
- Suggestions for improvement

---

## 🛠️ Technologies Used
- Python
- Flask
- PyMuPDF (fitz)
- python-docx
- Regex
